# Mesto 

Сервис, где люди делятся фотографиями интересных мест

## В работе использовались

- HTML 
- CSS 
- JS(ES6)
- БЭМ
- API
- Webpack

## Функции страницы

- редактировать профиль
- добавлять карточки
- удалять карточки
- ставить лайки
- открывать фотографии на весь экран
- валидация форм при помощи JS и HTML
